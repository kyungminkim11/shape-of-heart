# 마음의 모양 · Shape of Heart

성인 애착을 고정된 네 가지 성격 유형이 아니라 **애착 불안**과 **애착 회피**의 두 차원으로 이해하도록 돕는 모바일 우선 웹앱입니다.

## 주요 기능

- 애착 불안·회피 두 축 인터랙티브 시각화
- 안정형·불안집착형·거부회피형·공포회피형 설명
- 교육용 자기점검 12문항
- 결과 로컬 저장 및 공유
- 추격–철회 악순환과 대화 가이드
- 연구 체계와 용어 구분
- 다크 모드·오프라인 캐시·반응형 UI
