# 마음의 모양 · Shape of Heart

성인 애착을 고정된 성격표가 아니라 **애착 불안**과 **애착 회피**의 두 차원으로 이해하도록 돕는 모바일 우선 정보 웹사이트입니다.

## 사이트 구성

- **홈 대시보드**: 주요 정보와 페이지를 한곳에서 탐색
- **유형별 특징**: 안정형·불안집착형·거부회피형·공포회피형 심층 분석
- **16가지 관계 조합**: 나와 상대의 방향성을 구분한 조합 탐색기
- **유형 테스트**
  - 단순형 12문항
  - 복잡형 36문항 및 6개 세부 반응 영역
- **애착 용어 사전**: 기초 이론, 조절 전략, 관계 기술, 연구 용어 검색·필터
- **애착 백과사전**: 개념의 역사, 측정 체계, 관계 악순환, 안정화 연습, 대화 예시
- **열린 프로젝트**: 공개 저장소를 통해 내용과 기능 개선 의견 수렴

## 설계 원칙

- 유형 라벨보다 불안과 회피의 연속적인 정도를 우선합니다.
- 아동 애착 관찰, 성인 연애 자기보고, 성인애착면접을 구분합니다.
- 애착 유형을 행동의 면죄부나 정신질환 진단으로 사용하지 않습니다.
- 테스트 문항은 교육용으로 자체 작성했으며 ECR-R 등 표준화 검사를 복제하지 않습니다.
- 결과와 진행 상태는 사용자의 브라우저에만 저장합니다.
